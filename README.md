# Games Web App

A React-based web application featuring multiple classic games.

## Available Games

- **Hangman** - Guess the word before running out of attempts!
- **Tic-Tac-Toe** - Get three in a row to win!

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173/`

### Build

Build for production:

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 19
- Vite
- CSS3
